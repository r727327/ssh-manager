# SSH Manager - Current Features

## SSH Terminal
- Connect to servers via SSH using password or private key.
- Send commands to the remote server.
- Receive real-time output from remote shell.

## SFTP / File Operations
- List files and directories on the remote server.
- Read and write files (limited to <5MB for editing).
- Upload and download files.
- Delete files and directories.
- Create directories and empty files.
- Rename files or directories.

## Server Management
- Add, update, delete server entries.
- Store server info locally using Electron Store.

## Local File System
- Browse local directories and files.
- Get file metadata (size, permissions, modified time).

## UI / Window Controls
- Minimize, maximize, and close window.
- Frameless window support.

## Native Dialogs
- Open file dialog.
- Save file dialog.

## Events
- Terminal output event subscription.
- Terminal disconnect event subscription.
